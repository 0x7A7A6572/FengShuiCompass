## 1. 快速开始

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装依赖

```bash
cd FengShuiCompass
npm install
```

---

## 2. 本地运行项目

### 运行主演示页面

```bash
npm run dev
```

访问 `http://localhost:5173`（或终端显示的地址），可以看到完整的罗盘演示页面，包含右侧控制面板。

### 运行爻组件示例

```bash
npm run example
```

访问 `http://localhost:5173`，查看爻组件的独立演示。

### 构建生产版本

```bash
npm run build        # 构建应用
npm run build:lib    # 构建 npm 库（用于发布到 npm）
```

## 4. 修改罗盘数据

### 方式一：直接修改主题文件

编辑 `src/themes/theme-compass.ts`，保存后 Vite 会自动热更新（无需手动刷新）。

### 主题配置结构

```typescript
const theme: FengShuiCompassConfig = {
  info: { id: 2, name: "一", preview: "theme-compass-preview.png" },
  animation: { enable: true, duration: 1000, delay: 100 },
  rotate: 0,                    // 罗盘整体旋转角度
  autoFontSize: false,         // 是否自动计算字体大小
  isShowScale: true,           // 是否显示刻度
  compassSize: { width: 800, height: 800 },  // 罗盘尺寸
  latticeFill: [[0, 3, "red"]], // 宫格填充：[层索引, 宫索引, 颜色]
  line: { borderColor: "#AAAAAA", scaleColor: "#AAAAAA", ... },
  isShowTianxinCross: true,    // 是否显示天心十字
  data: [ ... ]                 // 核心数据，7 层罗盘数据
};
```

### data 数组 — 7 层罗盘数据

| 层索引 | 名称 | 数据示例 |
|--------|------|----------|
| 0 | 八数 | `["一", "二", "三", ...]` |
| 1 | 后先天八卦 | `[["坎","艮",...], ["☰","☲",...], ...]` (3组数据) |
| 2 | 九星 | `["贪", "巨", "禄", ...]` |
| 3 | 二十四山 | `["子", "癸", "丑", ...]` (24项) |
| 4 | 微盘二十四星 | 24项 |
| 5 | 透地六十龙 | 60项（垂直显示） |
| 6 | 透地六十龙旺相 | 60项 |

### 示例：修改第 0 层数据

```typescript
data: [
  {
    name: "八数",
    startAngle: 60,
    fontSize: 18,
    textColor: "white",
    vertical: false,
    togetherStyle: "empty",
    data: ["一", "二", "三", "四", "五", "六", "七", "八"],  // 修改这里
  },
  // ... 其他层
],
```

### 示例：修改第 2 层文字颜色

```typescript
{
  name: "九星",
  startAngle: 0,
  textColor: "red",  // 改为红色
  data: ["贪", "巨", "禄", "文", "武", "廉", "破", "辅", "弼"],
},
```

## 5. 主题配置详解

### 基础参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `rotate` | Number | 0 | 罗盘旋转角度（0-360） |
| `compassSize.width` | Number | 800 | 罗盘宽度 |
| `compassSize.height` | Number | 800 | 罗盘高度 |
| `isShowScale` | Boolean | true | 是否显示刻度 |
| `isShowTianxinCross` | Boolean | true | 是否显示天心十字 |

### 层配置参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `name` | String/Array | 层名称，数组时可配置多层标题 |
| `startAngle` | Number | 起始角度（0-360） |
| `fontSize` | Number | 字体大小 |
| `textColor` | String/Array | 文字颜色，支持数组为不同宫格设置不同颜色 |
| `vertical` | Boolean | 是否垂直显示文字（沿径向） |
| `togetherStyle` | String | `"empty"` 常规 / `"equally"` 等分 |
| `shape` | String | `"circle"` 圆形 / `"polygon"` 多边形 |

### 宫格填充 (latticeFill)

格式：`[[层索引, 宫索引, 填充颜色], ...]`

```typescript
latticeFill: [
  [0, 3, "red"],    // 第0层第3宫填充红色
  [1, 5, "#00FF00"] // 第1层第5宫填充绿色
],
```

## 6. 在控制面板实时调试

运行 `npm run dev` 后，页面右侧有一个**控制面板**，可以直接修改样式并实时预览。

### 控制面板功能

- **旋转罗盘**：拖动滑块旋转罗盘
- **罗盘大小**：调整宽度、高度、天池半径
- **样式修改**：选择层和宫，调整颜色
- **边框与刻度**：调整颜色和显隐
- **主题选择**：切换预设主题
- **🔑 罗盘渲染数据**：底部 JSON 编辑器，修改后点击「更新」

### JSON 编辑器使用

编辑器中包含完整的 theme 对象（包含 `info`、`data`、`compassSize` 等所有属性）。

**修改示例**：修改第 2 层"九星"的文字

```json
{
  "info": { "id": 2, "name": "一" },
  "data": [
    // ... 第0层、第1层数据
    {
      "name": "九星",
      "startAngle": 0,
      "textColor": "red",   // ← 修改这里
      "data": ["贪", "巨", "禄", "文", "武", "廉", "破", "辅", "弼"]
    },
    // ... 其他层
  ]
}
```

点击「更新」后，罗盘会立即刷新显示修改后的效果。

## 7. 在其他 Vue 3 项目中如何使用组件

### 方式一：npm 安装（推荐）

```bash
npm install feng-shui-compass
```

### 在 Vue 项目中使用

```vue
<script setup>
import { ref } from 'vue'
import { FengShuiCompassSvg } from 'feng-shui-compass'
import 'feng-shui-compass/dist/style.css'

// 罗盘配置
const compassConfig = ref({
  compassSize: { width: 800, height: 800 },
  rotate: 0,
  data: [
    {
      name: "八数",
      startAngle: 60,
      fontSize: 18,
      textColor: "white",
      vertical: false,
      togetherStyle: "empty",
      data: ["一", "二", "三", "四", "五", "六", "七", "八"],
    },
    // ... 其他层配置
  ]
})

// 处理宫格点击
function handleLatticeClick(event) {
  console.log('点击了宫格：', event)
}
</script>

<template>
  <FengShuiCompassSvg
    :config="compassConfig"
    @latticeClick="handleLatticeClick"
  />
</template>
```

### 方式二：从源码构建库

```bash
# 1. 构建库
npm run build:lib

# 2. 会在 dist/ 目录生成
#    - feng-shui-compass.es.js (ES module)
#    - feng-shui-compass.umd.js (UMD)
#    - types/ (类型声明)

# 3. 复制 dist 目录到你的项目，或发布到私有 npm
npm run publish:npm
```

### 方式三：直接复制组件源码

如果你只需要部分功能，可以直接将 `src/components/feng-shui-compass-svg/` 目录复制到你的项目中使用。

### 组件属性说明

| 属性 | 类型 | 说明 |
|------|------|------|
| `config` | FengShuiCompassConfig | 完整的罗盘配置对象 |
| `config.compassSize.width` | Number | 罗盘宽度 |
| `config.compassSize.height` | Number | 罗盘高度 |
| `config.rotate` | Number | 旋转角度 |
| `config.data` | Array | 层数据配置 |
| `config.isShowTianxinCross` | Boolean | 显示天心十字 |
| `config.line.borderColor` | String | 边框颜色 |

### 事件

| 事件名 | 参数 | 说明 |
|--------|------|------|
| `latticeClick` | `{ layerIndex, latticeIndex }` | 宫格点击事件 |

## 常见问题

### Q: 修改数据后页面没有更新？

确保保存了文件。Vite HMR 通常会自动热更新，如果没有，尝试手动刷新浏览器。

### Q: JSON 编辑器报错"必须包含 data 数组"？

编辑器的 JSON 内容应该是完整的 theme 对象，不是单独的 data 数组。确保最外层有 `data` 字段。

### Q: 如何添加新的一层？

在 `data` 数组末尾添加新的层对象：

```typescript
{
  name: "新层名称",
  startAngle: 0,
  fontSize: 16,
  textColor: "white",
  vertical: false,
  data: ["项1", "项2", ...],
},
```

### Q: 如何让文字垂直显示？

设置 `vertical: true`，文字会沿径向方向垂直显示。
